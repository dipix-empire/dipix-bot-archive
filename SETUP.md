Bot uses environment vars, support .env file in project root

```env
TOKEN="BOT TOKEN"
PTERODACTYL="PTERODACTYL TOKEN"
PANELHOST="PTERODACTYL HOST URI"
PANEL_ID="PTERODACTYL SERVER ID"
CLIENT="BIT CLIENT SERCET"

FORMS_CHANNEL_ID=""
ADMIN_ROLE_ID=""
PLAYER_ROLE_ID=""
GUEST_ROLE_ID=""
```
